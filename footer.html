Media object

the media object is a common pattern in web design. it has an img or other element to one side and text to the right. with flexbox we can allow the part of the media object containing the image to take its sizing info from the img and then the body of the media object flexes to take up the remaining space.

This is the content of my media object. items directly inside the flex container will be aligned to flex-start

to prevent the image from growing too large add a max-width to the image

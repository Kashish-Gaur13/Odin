Alignment

How to align items inside a flex conatiner both vertically and horizontally.

If we want items to stay the ssme width but distribyte themselves differently then we remove flex: 1 from .item(Or any class) and add justify-content: space-between to .container.

justify-content aligns items across the main axis. justify-content:center; aligns the items to center along hte main axis.

GAP property

setting gap on a flex container simply adds a specified space between flex items. its a new property so it doesnt show up many resources yet. gap:8px;

the main axis

the main axis is defined by flex-direction which has 4 values: row, row-reverse, column, column-reverse.

The flex container

an area of a Document laid out using flexbox is called a flex container display:flex or inline-flex;

the flex flow shorthand

we can combine the two properties flex direction anf flex wrap into the flex flow shorthand. the first value specified is flex direction and the second value is flex wrap. flex-flow: row wrap; flex-flow: column nowrap;

the flex basis property

it is what defines the size of the item in terms of space ot ;eabes as available space. initial value is auto

The flex grow property

with the flex-grow property set to a positive integer, flex items can grow along the main axis from their flex-basis. causes the item to stretch and take up any available space on that axis. if we gave all our items a flex-grow value of 1 then the available space would be equally shared btwn our items. if we give our first item a value of 2 and others a value of 1, 2 parts of the available space will be given to the first item and 1 part to the rest.

The flex-shrink property

it controls how space is taken away. if we dont have enough space and flex-shrink is set to a positive integer then the item can become smaller than the flex-basis. an item with a higher value set for flex-shrink will shrink faster than its siblings that have lower values.

predefined shorthand values

  • flex: initial - resets the item to the initial values of flexbox. same as fkex: 1 0 auto
  • flex: auto - same as flex: 1 1 0. items can grow and shrink as required
  • flex: none - creates fully inflexible flex items. same as flex: 0 0 suto
  • flex: < positive-number >

the align items property

the align-items property will align the items on the cross axis initial value of this property is stretch. we could aslo set align-items to flex-start in order to make them line up at the start of the flex container. flex-end to align them to the end, or center to align them in the center

justify-content

used to align the items on the main axis, the direction in which flex-direction has set the flow. the inital value is flex-start which will line the items up at the start edge of the container, but u could also setthe value to flex-end to line them up at the end or center to line them up in the center

you can also use space-between to take all the spare space after the items items have been laid out, and hsare it evenly betwn the items, so that there is an equal amount of space between each item

to create an equal amount of space on the right and left of each item, use space-around. items have a half-sixe space on either end

to cause items to have equal space around them use the value space evenly. items have a full size space on either end

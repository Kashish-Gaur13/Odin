THE BOX MODEL

Every single thing on webpage is a rectangular box. these boxes have other boxes in them and can sit alongside one another

ways to manipulate the size of these boxes and the space between them, using padding, margin and border.

  • padding: increases space btwn edge of a box and the content inside of it
  • margin: increases the space betwn a box and any others that sit next to it
  • border: adds space between margin and padding

Block and Inline boxes

In CSS we have two types of boxes-

  • Block
  • Inline
Boxes have an inner display type and an outer display type.

We can set various values for the display type using the display property.

Outer display porperty

if a box has an outer dispaly type of block, then:

  • The box will break onto a new line
  • The iwdth and height properties are respected
  • Padding, margin and border will cause other elements to be pushed away from the box.
  • The box will extend in the inline direction to fill the space available in its container. In most cases, the box will become as wide as its container, filling up 100% of the space available.
ex: h1, p, use block as their outer display type by default

If a box has an outer display type of inline, then:

  • The box will not break onto a new line.
  • The width and height properties will not apply.
  • Vertical padding, margins, and borders will apply but will not cause other inline boxes to move away from the box.
  • Horizontal padding, margins, and borders will apply and will cause other inline boxes to move away from the box.
    • Some HTML elements, such as a,span, em and strong use inline as their outer display type by default.

Inner display type

dictates how elements inside a box are laid out. You can change the inner display type for example by setting display: flex;. The element will still use the outer display type block but this changes the inner display type to flex. Any direct children of this box will become flex items

Axes of a flex container

  • 2 axes of flex container?
  • change those axes to arrange content in columns instead of rows?
the default direction for a flex container is Horizontal, or row but we can change the direction to vertical, or column: css .flex{ flex-direction: column;}

Axes

flex conta is thought to have 2 axes: the main axis and the cross axis. it is the direction of these axes that changes when the flex-direction is changed. flex-direction:row puts the main axis horizontal(L-R) flex-direction:column puts the main axis vertical(T-B)

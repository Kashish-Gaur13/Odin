About CSS


At the most basic level, CSS is made up of various rules.


These rules are made up of a selector and a semi colon separated list of declarations with each of those declarations being made of a property:value pair


DIV


div tag is one of the basic html elements. it is simply an empty container and divides ur webpage into different modifiable sections


Selectors


Theyre what is actually being selected for each rule.


Universal selector

will select elements of any type. syntax is a simple asterisk followed by {}


*{ color : purple; }

Type selectors

selects elements of the given type

div { color : white; }

Class selectors

selects elements with the given class name. Class is one of the attributes u place on an html element

.class_name{ color : red; } note that classes arent unique

ID Selectors

select element with the given ID. ids are unique

#id_name { color:red; }

Grouping Selector

For group of elements that shar some of their style declarations

.class_name , class_name {
similar declarations :
}

Descendant Combinator
